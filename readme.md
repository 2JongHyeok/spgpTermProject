# SMART_PHONE_GAME_PROGRAMMING_TERM_PROJECT

스마트폰 게임 프로그래밍 텀 프로젝트입니다.

## :star2: 프로젝트 제목

용사님 키우기 !

## :iphone:게임 간단 요약

스테이지를 클리어하여 장비와 동료를 얻고 내 영웅을 강화하는 게임입니다.



## :page_with_curl:게임 설명

버려진 아기 영웅이 성장해 나아가며 동료를 모아 보스를 처치하는 게임 입니다.

플레이어들은 아기 영웅으로 게임을 시작하게 됩니다. 

스테이지를 클리어 해 가며 재화를 얻게 되고,

얻은 재화로 음식과, 장비, 동료를 구매할 수 있습니다.

아기와 동료들을 성장시켜 나쁜 보스를 죽이세요 !!



## :video_game: 조작법 

* 기본 이동

  조이스틱을 사용하여 주인공 이동

* 공격

  주인공과 동료들은 움직이면서 일정 시간마다 자동으로 공격 함

* **사용된 기술**
  1. 오토타겟팅 : 적이 일정 거리 안으로 들어오면 플레이어가 자동으로 총알을 발사
  2. 플레이어 자동 추적 : 적은 항상 플레이어를 추적하여 따라옴
    
* **참고한 것**   
   수업시간에 배운 내용

* **수업내용에서 차용한 것**
  1. 씬 전환
  2. 액티비티 전환
  3. 일부 리소스

* **직접 개발한 것**
  1. 오토 타겟팅
  2. 플레이어 자동 추적 ( 오토타겟팅과 유사 )

* **아쉬웠던 부분**
  1. Android Studio라는 툴과 JAVA언어가 익숙하지 않아서 코딩을 할때 매우 힘들었다.
  2. 메인화면, 상점화면, Hero 화면을 밑에 있는 버튼들로 이동 가능하게 하고 싶었지만 하지 못하였다.
  3. 상점 페이지 보충, 몬스터 생성 알고리즘 보충, 수직 맵 뿐만 아니라 수직 + 수평 확장맵 사용 필요, 다양한 무기 필요
  4. 총알을 발사할때 특정 위치에서 총알이 커지는 버그를 수정하지 못하였다.
